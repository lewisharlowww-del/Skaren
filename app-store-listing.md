# Skaren — App Store Listing (ASO-optimized)

> **How Apple ASO works:** The App Store indexes your **App Name (30 chars)**,
> **Subtitle (30 chars)**, and **Keyword field (100 chars)** for search. These
> three fields should share **no repeated words** (Apple combines them, so
> repeats waste space). Apple auto-handles plurals, casing, and spaces, so the
> keyword field should be lowercase, comma-separated, **no spaces**. The
> description is **not** indexed for search but drives conversion. Character
> counts below are verified but re-check in App Store Connect before publishing.
